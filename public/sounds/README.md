# Sounds Directory

Place your audio files here:

## Required Files:

### Music (Background):
- `background-music.mp3` - Background music for learning games (loopable)

### Sound Effects:
- `correct.mp3` - Sound when answer is correct
- `wrong.mp3` - Sound when answer is wrong
- `level-up.mp3` - Sound when player levels up
- `badge-earned.mp3` - Sound when badge is earned
- `streak.mp3` - Sound for streak milestones (5, 10, 25, 50, etc.)
- `button-click.mp3` - Sound for button clicks
- `game-start.mp3` - Sound when game starts
- `game-over.mp3` - Sound when game ends

## File Format:
- Use MP3 format for best compatibility
- Keep file sizes reasonable (under 500KB per file)
- Use 44.1kHz sample rate
- Mono or stereo both work

## Notes:
- If files are missing, the game will continue without sounds (graceful degradation)
- Users can mute sounds/music in settings
